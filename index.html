About the Game

Dive into a 2D grid-based world where you control a character navigating through various challenges. The game is built on a tile system, each having distinct properties such as solidity, bounce factor, friction, and gravity effects. These properties dynamically influence the player's movement, offering a unique gameplay experience with each tile interaction.The game provides mechanisms for the player to move left, right, and jump. Moreover, the game's view adjusts based on the player's position, ensuring you always have the best perspective of the action.Rendered on a canvas, this game blends classic 2D gaming with modern design aesthetics. Use the arrow keys to move around and experience the dynamic world tailored by varying tile properties.

The Clarity Engine

The game is powered by the Clarity engine, a custom-built game engine tailored for this 2D adventure. Key features and functions of the Clarity engine include:

Core Functions:

  • Clarity(): The heart of the engine, this function initializes the game's core elements.
  • error(message): Ensures smooth gameplay by handling and logging errors.
  • log(message): A utility function for logging messages or events during gameplay.
  • set_viewport(x, y): Dynamically sets the game's viewport dimensions for an optimal viewing experience.
  • load_map(map): Transforms game data into a playable map, setting the stage for your adventure.
  • get_tile(x, y): Fetches specific tile properties based on its coordinates, influencing gameplay dynamics.
  • update_player(): Continuously updates the player's position and state, based on interactions and environment.

Properties:

  • camera.x & camera.y: Determines the camera's position, ensuring the player always remains in focus.
  • tile_size: Standardizes the size of each tile, creating a uniform game grid.
  • limit_viewport: Controls whether the viewport is confined to the map boundaries or can extend beyond.
  • player.loc.x & player.loc.y: Tracks the player's position, crucial for interactions and game progress.

Key Aspects:

The game intricately blends various aspects to deliver a captivating gameplay experience:

  • Map: A 2D grid that lays out the game's world and challenges.
  • Tile: The building blocks of the game, each with distinct properties that influence gameplay.
  • Player: You, the adventurer, navigating through the game world.
  • Camera & Viewport: Adjusts the game view based on the player's position, ensuring an optimal perspective.
  • Gravity: Affects how the player moves, jumps, and interacts with the environment.
